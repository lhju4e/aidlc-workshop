# Functional Design Plan - Unit 3: Admin+Kitchen Frontend

## Unit 정보
- **이름**: frontend/admin
- **유형**: React SPA (TypeScript)
- **책임**: 관리자 대시보드 + 주방 디스플레이 UI
- **관련 Story**: US-06, US-07, US-08, US-09, US-10, US-11

## Plan Steps

- [x] Step 1: 관리자 인증 UI 컴포넌트 설계 (LoginPage, RegisterPage)
- [x] Step 2: 대시보드 주문 모니터링 UI 설계 (DashboardPage, 테이블 카드 그리드)
- [x] Step 3: 테이블 상세/세션 관리 UI 설계 (TableDetailModal)
- [x] Step 4: 메뉴 관리 UI 설계 (MenuManagementPage, 드래그앤드롭)
- [x] Step 5: 주문 내역 조회 UI 설계 (OrderHistoryPage)
- [x] Step 6: 주방 디스플레이 UI 설계 (KitchenDisplayPage)
- [x] Step 7: 공통 컴포넌트 및 상태 관리 설계
- [x] Step 8: Business Rules 정의 (프론트엔드 검증, 상태 전이, 알림)
- [x] Step 9: Domain Entities 정의 (프론트엔드 타입/인터페이스)
